# EmoAgent

> 一个刚开始构思的情绪感知 Agent 实验项目。

EmoAgent 现在还处在很早期的阶段。

它还不是一个成熟框架，也还没有确定最终的技术选型、工程边界和产品形态。比起过早定义一整套宏大的架构，现阶段更重要的是先把一个最小可运行的 Harness 搭出来，看看这条路是否成立。

## 这是什么

EmoAgent 想探索一件事：

能不能做一个既有陪伴感、有活人感、又有执行能力的 Agent。

它不只是响应指令，也不只是扮演情绪化角色，而是尝试把情绪理解-模拟、记忆存储、上下文管理和任务执行放进同一个持续演化的系统里。

## 现在在做什么

- 做一个基础Agent原型
- 探索是否值得继续扩展成更完整的 EmoAgent

## Roadmap

- [ ] **Phase 0**：根据 [learn-claude-code](https://github.com/shareAI-lab/learn-claude-code) 构建一个最小的 Harness

## 灵感来源


这个项目目前主要受到 [learn-claude-code](https://github.com/shareAI-lab/learn-claude-code) 和[AstrBot](https://github.com/AstrBotDevs/AstrBot)的启发。

与其说是在“实现智能”，不如说是在认真搭建一个适合智能工作的环境。同时，智能也不应该只是冰冷的执行工具。EmoAgent 也会从这里出发，先把最小 Harness 做出来，再决定后面怎么继续长。
